# opencreature

Animated terminal creatures. Standalone toy. Doesn't talk to OpenCode.

```bash
npx opencreature              # rotate all
npx opencreature raccoon      # specific creature
npx opencreature --no-loop    # one frame cycle, exit
```

`q` or `Esc` to quit.

## What

Four animated ASCII creatures (raccoon, troll, ogre, pigeon) — eyes blink, mouths move, wings flap. Built with React + Ink.

For the OpenCode plugin versions (persona inside the agent), see the sibling packages in this repo.
